# About This Project
This project is a back end for sumber bening cashier app

# How To Run 
Make file in root directory called `config.json` and copy this
```
{
    "user": [{
        "username": "admin",
        "Password": "123katasandi123"
    }],
    "secretKey": "Sumber_Bening_Secret_Key"
}
```
change your username, password and secretkey value and run:\
`npm start` for production mode.\
`npm run dev` for development mode.
